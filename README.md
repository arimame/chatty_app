Chatty App
=====================

A single page chatting app that uses React and Websockets. Users can send messages to each other in real time.

### Final Project


### Dependencies

##Client Side

*babel-core
*babel-loader
*babel-preset-es2015
*babel-preset-react
*css-loader
*node-sass
*sass-loader
*sockjs-client
*style-loader
*webpack
*webpack-dev-server
*react
*react-dom

##Server Side
*express
*ws
*uuid


### Getting Started

*Install all dependencies (using the "npm install" command)
*Run the development web server using the npm start
